HTML

From Wikipedia, the free encyclopedia

".htm" and ".html" redirect here. For other uses, see HTM

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a webpage semantically and originally included cues for its appearance.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded. HTML provides a way to create structured documents using elements like headings, paragraphs, lists, links, and quotes. Tags such as <img> and <input> insert content directly, while others like <p> and </p> provide structure. Browsers don’t display the tags, but use them to interpret the content.

HTML can embed scripts like JavaScript that affect page behavior. The World Wide Web Consortium (W3C), which maintains HTML standards, has encouraged using CSS for layout since 1997. HTML5 supports video and audio via the <canvas> element with JavaScript.

History

Development

In 1980, physicist Tim Berners-Lee, while at CERN, proposed and prototyped ENQUIRE, a system to share documents. In 1989, he wrote a memo proposing an Internet-based hypertext system. He defined HTML and built the first browser and server software in 1990.

The first HTML description was "HTML Tags", mentioned online by Berners-Lee in 1991. It described 18 elements, most based on CERN SGML. Eleven of these still exist in HTML 4.

HTML is a markup language used by browsers to render web pages. Default behaviors are defined by the browser, customizable with CSS. Many concepts originated in older formatting tools like RUNOFF and CTSS.

HTML was formalized as an IETF application of SGML in 1993, with proposals from Berners-Lee and Dan Connolly. Dave Raggett's competing draft added features like tables and forms.

After early drafts expired, the IETF created an HTML Working Group. In 1995, HTML 2.0 became the first formal standard. Further development shifted to the W3C in 1996. HTML became an international standard in 2000. HTML5 was developed from 2004, completed in 2014.

HTML Version Timeline

HTML 2

November 24, 1995

HTML 2.0 was published as RFC 1866. Supplemental RFCs added:

  • Nov 25, 1995: RFC 1867 (file upload)
  • May 1996: RFC 1942 (tables)
  • Aug 1996: RFC 1980 (image maps)
  • Jan 1997: RFC 2070 (internationalization)

HTML 3

January 14, 1997

HTML 3.2[16] was published as a W3C Recommendation. It was the first version developed and standardized exclusively by the W3C, as the IETF had closed its HTML Working Group on September 12, 1996. [17]

Initially code-named "Wilbur", [18] HTML 3.2 dropped math formulas entirely, reconciled overlap among various proprietary extensions, and adopted most of Netscape's visual markup tags. Netscape's blink element and Microsoft's marquee element were omitted due to a mutual agreement between the two companies. [14] A markup for mathematical formulas similar to that of HTML was standardized 14 months later in MathML.

HTML 4 Timeline

HTML 4

December 18, 1997
HTML 4.0[19] was published as a W3C Recommendation. It offers three variations:

  • Strict, in which deprecated elements are forbidden
  • Transitional, in which deprecated elements are allowed
  • Frameset, in which mostly only frame related elements are allowed
Initially code-named "Cougar",[18] HTML 4.0 adopted many browser-specific elements from Netscape and Internet Explorer, but also sought to phase out Netscape's visual markup features by marking them as deprecated in favor of style sheets. HTML 4 is an SGML application conforming to ISO 8879 – SGML.[20]

April 24, 1998
HTML 4.01[21] was reissued with minor edits without incrementing the version number.

December 24, 1999
HTML 4.01[22] was published again with three variations. The same three variations as HTML 4.0 and its last errata[23] was published on May 12, 2001.

May 2000
ISO/IEC 15445:2000[24] ("ISO HTML", based on HTML 4.01 Strict) was published as an ISO/IEC international standard.[25] It is documented in the domain of the ISO/IEC JTC 1/SC 34 (ISO/IEC Joint Technical Committee 1, Subcommittee 34 – Document description and processing languages).[26]

After HTML 4, there were no new versions of HTML for many years, as the development of the parallel, XML-based language XHTML occupied the W3C's HTML Working Group.
